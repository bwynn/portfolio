flexbox

This css property has been getting a lot of attention in blogs lately, and I've been playing around with it as it gains wider browser support. Vendor prefixing covers most major browsers, and very easy to implement good looking layouts in short order, I see this as the future of display templates in css. Browsing around any of my code, you'll likely come across quite a bit of this property.

Browser Plan

  • Safari 8/ OS X 10.10
  • Chrome 42/ OS X 10.10
  • Firefox 39/ OS X 10.10
  • Chrome 42/Win 7
  • IE 11/Win 8.1
